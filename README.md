# DaD

Вы, смайлик, который блуждает по полю в поисках финиша и монет. Ваша цель: Дойти до финиша, собрав все монетки и обойдя коробки (то есть ваши препятствия)
P. S. Будьте осторожны, ведь рандом здесь не знает границ, и поэтому вы можете быть окружены коробками XD
